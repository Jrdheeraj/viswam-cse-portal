# Viswam Engineering College - CSE Department Portal

A high-end, interactive web application developed for the **Computer Science and Engineering (CSE)** department of Viswam Engineering College. This portal is designed with a premium, modern aesthetic, featuring glassmorphism, custom CSS animations, and a visionary bi-column layout.

## 🚀 Key Features

### 🎓 Visionary Leadership
- **HOD Profile**: A dedicated, content-rich section for Dr. R. Vasanthselvakumar, featuring a bi-column layout, detailed three-paragraph bio, and structured academic credentials with hover-triggered image scaling and card lifting.

### 📚 Academic Framework
- **PEOs & PSOs**: Redesigned as interactive, bi-sectioned cards inspired by premium design aesthetics (CodeDale). 
- **Themed Visuals**: Specialized top visuals for each objective, including **floating binary particles** (Foundation), **pulsing lightbulbs** (Innovation), and **dynamic network nodes** (Problem Solving).

### 🏆 Placement Excellence
- **Live Stats**: Pulsing "Offers Count" and bidirectional, professional-grade marquees for top recruiters.
- **Trophy Container**: Glassmorphic, pulsing trophy visual highlighting the department's highest packages.

### 🌟 Department Activities
- **Attractive Animations**: Unique, theme-based background animations for Workshops (circuit nodes), Hackathons (binary bits), Research (pulsing rings), and Innovation (stardust fields)—all maintained within a clean, consistent card structure.

### 📱 Responsive Experience
- **Mobile-Optimized**: Fully responsive layout with a modern, pill-shaped mobile navigation menu for seamless usage on any device.

## 🛠️ Technology Stack
- **Frontend**: React (TSX), Vite, Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Custom Vanilla CSS Keyframes & Framer Motion transitions (simulated via CSS)
- **Styling**: Glassmorphism, Premium Gradients, and Soft Airy Shadows

## 📦 Getting Started

1. **Clone the repository**:
   ```bash
   git clone [repository-url]
   cd viswam-cse-portal
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run in Development Mode**:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```

## 📁 Project Structure
- `src/components`: Clean, modular TSX components for each department section.
- `src/index.css`: Global design system, utility classes, and optimized keyframe animations.
- `public/`: Static assets (HOD photos, recruiter logos, etc.).

## ✨ Design Principles
- **Aesthetic First**: Focus on "wow" factor with vibrant colors and subtle micro-interactions.
- **Content Scannability**: Use of bold headers and clear, structured lists for academic excellence.
- **Performance**: Lightweight animations and optimized build sizes for fast-loading college portals.
